Slurm-web

CAUTION: Please note that this version of Slurm-web is not maintained anymore. A new roadmap has been defined with future plans.

Slurm-web is a web application that serves both as web frontend and REST API to a supercomputer running Slurm workload manager. It is a free software licensed under the GPLv3. Read the introduction for more details.
